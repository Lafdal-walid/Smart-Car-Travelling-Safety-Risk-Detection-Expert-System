# نظام تقييم مخاطر القيادة

واجهة ويب تفاعلية لنظام تقييم مخاطر القيادة باستخدام نظام الخبير.

## المميزات

- واجهة مستخدم عربية حديثة وسهلة الاستخدام
- ثلاث فئات للحقائق: سيارة، سائق، عوامل خارجية
- تقييم تلقائي للمخاطر باستخدام نظام الخبير
- توصيات مفصلة بناءً على التحليل
- تصميم متجاوب يعمل على جميع الأجهزة

## المتطلبات

- Python 3.7+
- المكتبات المذكورة في `requirements.txt`

## التثبيت والتشغيل

1. تثبيت المكتبات:
```bash
pip install -r requirements.txt
```

2. تشغيل الخادم:
```bash
python app.py
```

3. فتح المتصفح على:
```
http://localhost:5000
```

## هيكل المشروع

- `index.html` - واجهة المستخدم الرئيسية
- `app.py` - خادم Flask Backend
- `main.py` - الملف الرئيسي لنظام الخبير
- `risk_expert_system.py` - نظام تقييم المخاطر
- `recommendation_expert_system.py` - نظام التوصيات
- `knowledge_base.py` - قاعدة المعرفة
- `reasoning_engine.py` - محرك الاستدلال
- `risk_rules.csv` - قواعد تقييم المخاطر
- `recommendation_rules.csv` - قواعد التوصيات

## الاستخدام

1. افتح الواجهة في متصفحك
2. املأ البيانات في الفئات الثلاث (سيارة، سائق، عوامل خارجية)
3. اضغط على "توليد التقييم"
4. شاهد مستوى المخاطر والتوصيات

## ملاحظات

- النظام يعمل مع وبدون اتصال بالخادم (يحتوي على محاكاة احتياطية)
- الواجهة باللغة العربية مع دعم RTL
- التصميم متجاوب ويعمل على الهواتف والأجهزة اللوحية
